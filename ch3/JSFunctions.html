JavaScript is a scripted or interpreted language that runs as-is in the browser.  Unlike some typical programming languages that are written in syntax close to English, then compiled to assembly or even machine code, JavaScript is saved to a web server as-is and called or run by the page when the html page loads.

How is it used?  Javascript works in collaboration with other web technologies, like html and css.  Think of the latest version of html5 like a three-legged stool.  Working together, they make rich, interactive pages and web sites, that entertain and inform users with content that changes as the needs of the user changes.  Think about how you might customize a web site or page.  Put in name, login, location, or other preferences, or search for elements within a shopping site to display custom results on a page based on what you asked.  This is the magic of dynamic, flexible web pages that can appear different to different users.

Java? No - Java is an object oriented program developed by James Gosling and team  starting in 1991 and released by Sun Microsystems in 1996.  Java is currently managed by Oracle.  Java needs to be compiled and does not need to be run through a browser.  so in short, JavaScript and Java are not the same.

What can I do with JavaScript? - Create user interactions, play games, draw animations and graphs, run calculations and simulations, and more.  3d? Yep.  CAD? Yep.   You name it.  If you saw it on the web, javascript probably played a part.
